# Outward

Privacy-first React Native + Expo mobile application for effort-based mental health tracking with offline-first local storage.

## 🌐 Live Demo

https://outward--fek18efspq.expo.app

## 💻 GitHub

https://github.com/ans25et/Outward

## Overview

Outward is a privacy-first mental health companion that helps users build healthy habits through effort-based progress tracking rather than mood alone. The app allows users to log daily recovery activities, record reflections, visualize trends, and monitor long-term progress while keeping all personal data stored locally on the device.

## Features

- 📱 Built with React Native and Expo
- 🔒 Offline-first local storage
- 📝 Reflection journaling
- 📈 Recovery progress tracking
- 📊 Personalized insights
- 📤 JSON data export
- 🌙 Modern mobile interface

## Tech Stack

- React Native
- Expo
- TypeScript
- AsyncStorage
- Expo FileSystem
- Expo Sharing

## Installation

```bash
git clone https://github.com/ans25et/Outward.git
cd Outward
npm install
npx expo start
```
